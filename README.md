## SandVideoPlayer

SandVideoPlayer is a custom AVplayer on iOS.

### CocoaPods

`pod 'SandVideoPlayer'`

## Reqirements
- Swift 3
- iOS 9.0+
- Xcode 8

## Features
- [x] Support play local and network
- [x] Support full screen 
- [x] Support replay media
- [x] Support slide forward and backward
- [x] Support next and previous 
- [x] Media cache
